<h1>TO PRINT SEQUENCE FROM 1 TO N FOR X TIMES</h1>

**INPUT:**<br>
_3 4_

**OUTPUT:**<br>
_1 2 3 1 2 3 1 2 3 1 2 3_


C code:
```
#include <stdio.h>
#include <conio.h>
void main(){
    int n, x;
    scanf("%d %d", &n, &x);
    for(int i=0; i<x ; i++){
        for(int j=1; j<=n ;j++){
            printf("%d ",j);
        }
    }
}
```
